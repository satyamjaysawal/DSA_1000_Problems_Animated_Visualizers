📝 Problem Statement

Given an integer n, return True if it is a power of two, otherwise return False. An integer n is a power of two if there exists an integer x such that n == 2^x.

🧠 Solving Approach (Bit Trick)

Use the elegant n & (n - 1) == 0 trick. Powers of 2 have exactly ONE bit set in binary. Subtracting 1 flips that bit to 0 and turns all lower bits to 1. AND-ing them gives 0.

🎯 Core Concepts

Edge case: Negative numbers and zero are NOT powers of 2. Only positive integers qualify.
Bit insight: 8 = 1000, 7 = 0111, 8 & 7 = 0000 ✓. Compare 6 = 110, 5 = 101, 6 & 5 = 100 ≠ 0 ✗.
Why it works: A power of 2 in binary is a single 1 followed by zeros. n-1 flips the pattern: 0 followed by 1s. No bit overlap → AND = 0.
Time O(1) Space O(1)
