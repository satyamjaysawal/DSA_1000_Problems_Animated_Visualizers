GT LEARN #12 🏛️ Integer to Roman MEDIUM Greedy Math String
num= exp: "MCMXCIV"

📝 Problem Statement

Roman numerals use seven base symbols: I=1, V=5, X=10, L=50, C=100, D=500, M=1000. Plus six subtractive pairs: IV=4, IX=9, XL=40, XC=90, CD=400, CM=900. Given an integer num in range 1 ≤ num ≤ 3999, return its Roman numeral representation.

🧠 Solving Approach

Greedy + denomination ladder. List all 13 values (7 base + 6 subtractive) in descending order. For each, repeatedly subtract from num while it fits, appending the symbol each time. Greedy works because the subtractive pairs close every gap that would force a worse choice.
🎯 Key insight: Subtractive pairs (CM, CD, XC, XL, IX, IV) are placed in the value list itself — so the greedy never gets stuck "needing to write 4 I's".
♻️ Loop pattern: Outer for over 13 denominations + inner while num >= value to consume that denomination as many times as possible.
🏁 Termination: When num == 0, all subsequent while checks fail → loop exits → return result.
⏱ Time: O(1) — bounded by 3999 💾 Space: O(1) output buffer
