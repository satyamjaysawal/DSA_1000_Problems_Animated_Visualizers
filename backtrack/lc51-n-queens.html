GT LEARN #51 ♛ N-Queens HARD Backtracking Constraint Sets
n= exp: 2

📝 Problem Statement

Place n queens on an n × n chessboard such that no two queens attack each other. Queens attack along the same row, column, or diagonal. Return all distinct configurations.

🧠 Solving Approach

Place queens row by row (one per row, so row-attack auto-prevented). Maintain 3 sets to track threatened lines: cols, diag1 = row - col (↘ diagonals), diag2 = row + col (↙ diagonals). Backtrack when no safe column exists.

🎯 Core Concepts

Diagonal Math: All cells on a ↘ diagonal share the same row - col. All cells on a ↙ diagonal share the same row + col.
Base Case: When row == n, all queens placed safely → record solution.
Backtrack: After exploring a column, remove queen from sets to undo state before trying next.
Time O(n!) Space O(n²)
