GT LEARN#282🎨 Expression Add OperatorsHARDBacktrackingString
num=target=exp: ["1+2+3","1*2*3"]

📝 Problem Statement

Given string num (digits only) and integer target, insert +, -, or * between digits so expression evaluates to target. No leading zeros in multi-digit operands.

🧠 Solving Approach

Backtracking — har position pe decide karo: kitne digits leke operand banaye, kaunsa operator lagaye. Multiplication ke liye prev track karo (undo + redo for precedence).

🎯 Core Concepts

Base Case: index == len(num) → agar value == target toh expression valid hai.
Transition: num[index..i] ko operand banao, 3 operators try karo: +curr, -curr, *curr.
Pruning: Leading zero → agar num[index]=='0' aur i>index toh multi-digit "0x" invalid, break.
Time O(N·4ᴺ)Space O(N)
