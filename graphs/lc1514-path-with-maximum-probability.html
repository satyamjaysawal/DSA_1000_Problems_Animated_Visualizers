GT LEARN #1514 🗺️ Path With Maximum Probability MEDIUM Dijkstra Heap
exp: 0.25
📝 Problem Statement
Undirected weighted graph mein n nodes hain. Har edge (u,v) ka succProb[i] probability hai successful traverse hone ka. Find maximum probability path from start to end. If no path exists, return 0.0.
🧠 Solving Approach
Modified Dijkstra using max-heap on probability. Probability multiplies along path (not adds), and we want the maximum. Python ka heapq min-heap hai, isliye probability ko negate karke push karte hain.
🎯 Key Insight
Edge weights add nahi hote — multiply hote hain. Aur shortest nahi, longest (max) probability chahiye. Toh negate karke min-heap ko max-heap bana lete hain.
⏭ Stale Skip
Heap se nikla node agar already better probability se settle ho chuka hai (p < prob[u]), toh skip — yeh stale entry hai.
🏆 Early Exit
Jab end node heap se pop hota hai, woh apni best probability ke saath nikalta hai (Dijkstra invariant). Turant return!
⏱ Time: O((V+E) log V) 💾 Space: O(V+E)
