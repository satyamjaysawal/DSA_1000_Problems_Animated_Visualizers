GT LEARN #200 🏝️ Number of Islands MEDIUM DFS Grid Flood-fill
grid= exp: 2

📝 Problem Statement

Given a 2D grid grid of '1's (land 🟨) and '0's (water 🟦), return the number of islands. An island is surrounded by water and formed by connecting adjacent lands horizontally or vertically (NOT diagonally). Edges of grid count as water boundary.

🧠 Solving Approach

Grid scan + DFS flood-fill — har cell pe jao, agar land mile aur unvisited ho, toh count++ aur DFS se uske saare connected land ko sink karo (mark as '0'). Aise har naye 'island start' pe ek baar count badhega.

🎯 Core Concepts & Base Case

Base Case: DFS mein agar (r,c) out-of-bounds ya cell '0' (water/visited) hai → return.
Transition: Land cell sink karo (grid[r][c] = '0'), fir 4 directions (up/down/left/right) recurse.
Pruning: Sink karke same island ko dobara nahi count karte — visited tracking ka sundar trick.
Time O(R·C) Space O(R·C)
