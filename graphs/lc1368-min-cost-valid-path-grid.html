📝 Problem Statement

Diya gaya m × n grid jahan har cell mein arrow hai: 1=right, 2=left, 3=down, 4=up. Cell pe pohochke us arrow ki direction mein follow karte hain. (0,0) se (m-1, n-1) tak ka valid path banane ke liye minimum number of cell-arrow modifications chahiye. Har modification cost 1.

🧠 Solving Approach

0-1 BFS with deque: Graph banao jahan har cell se 4 neighbors connected hain — agar cell ka arrow neighbor ki taraf point karta hai, edge weight = 0; warna 1. Deque use karo: 0-cost edges ke liye appendleft, 1-cost edges ke liye append. Yeh Dijkstra ka special O(m·n) version hai!

🎯 Core Concepts

Free EdgeCell ka arrow already neighbor ki taraf — cost 0, deque ke front mein.
Cost EdgeArrow ulta hai — modify chahiye, cost 1, deque ke back mein.
RelaxationNaya path shorter? Toh dist[nr][nc] update + deque mein push.

⏱ Complexity

Time: O(m·n) Space: O(m·n)
