grid=exp: 1

📝 Problem Statement

Given a binary grid, every 1 is land and every 0 is water. Find the minimum number of days to disconnect the island. Ek din mein exactly ek land cell ko water bana sakte hain.

🧠 Solving Approach

Pehle current island count nikaalo. Agar already disconnected → 0. Warna har land cell ko temporary cut karke island count check karo; agar kisi ek cut se disconnect ho jaye → 1, warna guaranteed answer 2.

🎯 Core Concepts & Base Case

Base Case: Agar current island count ≠ 1, grid already disconnected → answer 0.
Transition: Har land cell ko temporarily remove karke helper DFS se island count check karo.
Key Insight: Answer hamesha 0, 1, ya 2 hota hai. 2 se zyada kabhi nahi lagta.
Time O((mn)²)Space O(mn)
