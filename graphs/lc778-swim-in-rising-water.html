GT LEARN #778 🌊 Swim in Rising Water HARD Dijkstra Min-Heap Grid
exp: 3
📝 Problem Statement
n×n grid mein har cell ki ek height hai. Time t pe water level t hai — tum sirf un cells mein swim kar sakte ho jinki height ≤ t hai. Start (0,0) se (n-1,n-1) tak pahunchne ka minimum time nikalo.
🧠 Solving Approach
Modified Dijkstra — heap pe priority = path ka maximum height seen so far (not sum!). Har neighbor ka cost = max(currentMaxHeight, neighborHeight). Min-heap se sabse "shallow" path nikalti hai.
🎯 Key Insight
Path ka cost = path mein aane wali sabse uchi cell. Sum nahi — bottleneck minimization!
🌊 Water Rising
Time t pe height ≤ t wali sab cells flooded hain — easy swim. Min path max-height = answer.
🏆 Early Exit
Jab end cell pop hota hai, woh apni best (min-max) probability ke saath nikalta hai. Turant return!
⏱ Time: O(n² log n) 💾 Space: O(n²)
