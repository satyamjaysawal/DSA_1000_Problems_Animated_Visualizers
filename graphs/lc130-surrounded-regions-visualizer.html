📝 Problem Statement

Given a 2D board of X and O, flip every O-region that is completely surrounded by X. Any O connected to the border must stay as O. Connection is only 4-directional: up, down, left, right.

🧠 Solving Approach

Seedha inner O flip karna error-prone hota hai. Better strategy: pehle border se connected saare safe O mark karo using DFS/stack, phir final pass mein jo O safe nahi hai usko X bana do.

🎯 Core Concepts & Base Case

Base Case: Agar board empty hai, answer wahi board hoga.
Transition: Border pe milne wala har naya O stack mein push hoga aur uske connected O safe mark honge.
Pruning: Jo cell X hai ya already safe mark ho chuka hai, usko dobara process nahi karte.
Time O(mn) Space O(mn)
