GT LEARN #743 📡 Network Delay Time MEDIUM Dijkstra Min-Heap Graph
exp: 2
📝 Problem Statement
Network of n nodes (1-indexed). times[i] = [u, v, w] means signal travels from u to v in w time. Source node k sends a signal — return min time for ALL nodes to receive it. If any node unreachable, return -1.
🧠 Solving Approach
Classic single-source shortest path → Dijkstra's algorithm. Min-heap pops nearest unsettled node, then relaxes all outgoing edges. Final answer = max of all dist values (slowest node decides total network delay).
🔑 Relaxation
For edge node → nei (w): if dist[node]+w < dist[nei], update dist[nei] and push to heap.
✂️ Stale-pop optimization
Same node may be pushed multiple times with different costs. When popped, if d > dist[node], this entry is stale — skip it.
🏁 Final Answer
max(dist.values()). If still inf → unreachable node exists → return -1.
⏱ Time: O((V+E)·log V) 💾 Space: O(V+E)
