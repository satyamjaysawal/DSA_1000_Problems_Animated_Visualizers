GT LEARN #241 🎨 Different Ways to Add Parentheses MEDIUM Divide & Conquer Recursion
expression= exp: [0, 2]

📝 Problem Statement

Diya gaya hai ek string expression jisme numbers aur operators (+, -, *) hain. Sare possible results return karo jo different parenthesization se mil sakte hain.

Example: "2-1-1" ko 2 tarah parenthesize kar sakte hain: (2-1)-1=0 ya 2-(1-1)=2. Toh answer = [0, 2].

🧠 Solving Approach

Divide & Conquer: Har operator ko split point maano. Operator ke left side ko recursively solve karo (saare possible values), right side ko bhi. Phir har (left, right) pair ke liye operator apply karke result combine karo.

🎯 Core Concepts

Base case: Agar expression mein koi operator nahi hai (pure number), toh [int(expr)] return karo.
Split: Har i pe agar expr[i] operator hai, toh left = expr[:i], right = expr[i+1:] bana ke recurse karo.
Combine: Cartesian product — har l from leftResults × har r from rightResults pe operator apply karo.
Time: O(Catalan(n)) Space: O(Catalan(n))
