📝 Problem Statement

Given an input string s and a pattern p, implement wildcard pattern matching with support for '?' and '*'. '?' matches any single character. '*' matches any sequence of characters (including the empty sequence). The matching should cover the entire input string.

🧠 Solving Approach

We use a 2D DP Table where dp[i][j] is true if the first i characters of s match the first j characters of p. We fill this table row by row. If we see a *, it can either match nothing (look left) or match one more char (look up).

Base Case Empty string matches empty pattern. * can match empty prefix.
Recurrence If chars match or ?, look diagonal. If *, look left or up.
Result Bottom-right cell tells us if full string matches full pattern.
Time: O(M*N) Space: O(M*N)
