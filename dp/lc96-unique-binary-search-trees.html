📝 Problem Statement

Given an integer n, return the number of structurally unique BSTs which has exactly n nodes of unique values from 1 to n. Constraints: 1 ≤ n ≤ 19.

🧠 Solving Approach

Classic Catalan number DP. Let dp[i] = number of unique BSTs with i nodes. For each root choice j in [1..i]: left subtree has j-1 nodes, right has i-j nodes → contributes dp[j-1] × dp[i-j].

🎯 Core Concepts & Base Case

Base Case: dp[0] = 1 (empty tree counts as 1 way) and dp[1] = 1.
Recurrence: dp[i] = Σ dp[j-1] × dp[i-j] for j in [1..i].
Insight: The structure count is independent of values — only depends on node count. So 1..3 has same count as 5..7.
Time: O(n²) Space: O(n) Catalan: C(n) = (2n)! / ((n+1)! · n!)
