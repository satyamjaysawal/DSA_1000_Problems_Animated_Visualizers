📝 Problem Statement

You have k identical eggs and a building with n floors. There's a critical floor f (0 ≤ f ≤ n) — eggs break if dropped from floor > f, and survive otherwise. Find the minimum number of moves needed to determine f with certainty in the worst case.

🧠 Solving Approach — Inverse DP

Instead of "min moves for n floors", flip it: "with k eggs and m moves, what's the MAX floors we can cover?" Then find smallest m where coverage ≥ n.

🎯 Core Recurrence

State: dp[i][m] = max floors solvable with i eggs and m moves.
Recurrence: dp[i][m] = dp[i-1][m-1] + dp[i][m-1] + 1
• If egg breaks: dp[i-1][m-1] floors below
• If egg survives: dp[i][m-1] floors above
• +1 for the floor we just dropped from
Stop when: dp[k][m] ≥ n — return that m.
Time: O(k·log n) Space: O(k·n) m ≤ n always
