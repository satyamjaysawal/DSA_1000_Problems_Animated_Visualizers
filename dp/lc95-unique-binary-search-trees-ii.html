📝 Problem Statement

Given an integer n, return all the structurally unique BSTs (binary search trees) which has exactly n nodes of unique values from 1 to n. Constraints: 1 ≤ n ≤ 8.

🧠 Solving Approach

For each value i in range [start, end], make i the root. Recursively build all possible left subtrees from [start, i-1] and all right subtrees from [i+1, end]. Combine each left × right pair under root i.

🎯 Core Concepts & Base Case

Base Case: When start > end → empty range → return [None] (one "empty tree" placeholder so combining loops still iterate).
Transition: For each root i: build(start, i-1) × build(i+1, end) → cartesian product of left × right subtrees.
BST property: All left descendants < root < all right descendants — guaranteed by splitting the range.
Time: O(n · Catalan(n)) Space: O(n · Catalan(n))
