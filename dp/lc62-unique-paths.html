📝 Problem Statement

A robot is at top-left corner of an m × n grid. The robot can only move down or right. How many unique paths exist from (0,0) to (m-1, n-1)?

🧠 Solving Approach

Build a 2D DP table where dp[i][j] = number of unique paths to reach cell (i,j). To reach any cell, robot must come from above or left → so dp[i][j] = dp[i-1][j] + dp[i][j-1].

🎯 Core Concepts & Base Case

Base Case: First row & first column = 1 (only one straight path along edge — all rights or all downs).
Recurrence: dp[i][j] = dp[i-1][j] + dp[i][j-1] for inner cells.
Math insight: Answer equals C(m+n-2, m-1) — choosing m-1 downs from m+n-2 total moves.
Time: O(m·n) Space: O(m·n) Math: C(m+n-2, m-1)
