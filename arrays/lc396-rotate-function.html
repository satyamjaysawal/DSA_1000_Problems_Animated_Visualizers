GT LEARN #396 🔄 Rotate Function MEDIUM Math · Greedy · O(n) exp: 26
📝 Problem Statement
Given integer array nums of length n, define F(k) = 0·arr_k[0] + 1·arr_k[1] + ... + (n-1)·arr_k[n-1] where arr_k is nums rotated clockwise by k positions. Return the maximum value of F(0), F(1), ..., F(n-1).
🧠 Solving Approach
Naive O(n²) — compute each F(k) directly. Smart O(n) — derive a recurrence: F(k) = F(k-1) + sum(nums) - n·nums[n-k]. Compute F(0) once, then roll forward in O(1) per step. Track the max.
🎯 Core Concepts & Recurrence
F(0): Σ i·nums[i] for i=0..n-1 (base case).
Rolling step: F(k) = F(k-1) + total - n·nums[n-k]. Each rotation pushes one element from the end to the front, changing its coefficient from (n-1) to 0 — a delta of total - n·nums[n-k].
Why it works: Every other element's coefficient increases by exactly 1, contributing total. The wrap-around element loses (n-1)·value, gaining 0·value — net change -n·nums[n-k].
⏱ Time: O(n) 💾 Space: O(1)
