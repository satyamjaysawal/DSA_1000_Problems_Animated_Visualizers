GT LEARN #2574 ⚖️ Left and Right Sum Differences EASY Prefix Sum Array
nums= exp: [15,1,11,22]

📝 Problem Statement

Given a 0-indexed array nums, return an array answer where answer[i] = |leftSum[i] − rightSum[i]|.

leftSum[i] = sum of nums[0..i−1] (zero if i=0). rightSum[i] = sum of nums[i+1..n−1] (zero if i=n−1).

🧠 Solving Approach

Compute total = sum(nums) once. Maintain a running left sum. For each i: right = total − left − nums[i], then answer[i] = |left − right|, then left += nums[i]. O(n) time, O(1) extra.

🎯 Core Concepts

Running Prefix: Instead of recomputing leftSum every iteration (O(n²)), maintain it incrementally — add nums[i] AFTER processing index i.
Total-Trick: rightSum = total − left − nums[i] avoids a second loop. Total stays constant.
Edge cases: i=0 → leftSum=0; i=n−1 → rightSum=0.
Time O(n) Space O(1)*
