rows= cols= rStart= cStart= expected: 12 cells
📝 Problem Statement
Tum ek R × C grid pe khade ho, starting cell (rStart, cStart). Tumhe spiral order mein chalna hai (East → South → West → North), aur grid ke har valid cell ko visit karna hai. Agar koi step grid ke bahar (out-of-bounds) chala jaaye, toh skip karo lekin direction aur movement count maintain rakho. Final answer = R*C cells in order of visit.
🧠 Solving Approach
Simulation karo with a step-doubling pattern. Pattern: 1 East, 1 South, 2 West, 2 North, 3 East, 3 South, 4 West, 4 North... — har 2 directions ke baad step count +1 hota hai. Out-of-bounds cells skip ho jaate hain par direction sequence chalti rehti hai.
🎯 Core Concepts
Direction CycleE(0,1) → S(1,0) → W(0,-1) → N(-1,0). After every 2 dirs, steps += 1.
TerminationLoop tab tak chalega jab tak len(result) < R*C. Saari valid cells visit ho jayein.
Bounds Check0 ≤ r < R and 0 ≤ c < C — yeh true hua tabhi cell append hogi.
⏱ Time: O(max(R,C)²) 💾 Space: O(R·C)
