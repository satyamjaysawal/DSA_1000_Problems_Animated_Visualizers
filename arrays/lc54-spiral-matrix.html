📝 Problem Statement

Given an m × n matrix, return all elements of the matrix in spiral order — starting from top-left, going right, then down, then left, then up, spiraling inward until every cell is visited exactly once.

🧠 Solving Approach

Maintain 4 boundary pointers: top, bottom, left, right. In each spiral layer, traverse 4 directions (→, ↓, ←, ↑) and shrink the corresponding boundary after each pass. Stop when boundaries cross.

🎯 Core Concepts & Base Case

Boundary: 4 walls shrinking inward Order: right → down → left → up Stop: top > bottom OR left > right Edge case: single row/col — extra checks before left/up traversal
⏱️ Time: O(m·n) 💾 Space: O(1) extra
