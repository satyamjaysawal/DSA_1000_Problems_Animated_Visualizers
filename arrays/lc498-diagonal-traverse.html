📝 Problem Statement

Given an m × n matrix mat, return all elements in diagonal order. Diagonals alternate direction — first goes ↗ up-right, next goes ↙ down-left, and so on.

🧠 Solving Approach

Simulation — total diagonals = m + n - 1. For each diagonal index d, all cells on it satisfy row + col = d. Even d → walk up-right; Odd d → walk down-left. Pick correct starting cell based on direction.

🎯 Core Concepts

Diagonal Math: All cells on diagonal d have r + c = d. Total diagonals = m + n - 1.
Even d (↗): Start at r = min(d, m-1), c = d - r. Walk: r--, c++.
Odd d (↙): Start at c = min(d, n-1), r = d - c. Walk: r++, c--.
Time O(m·n) Space O(1)
