GT LEARN #217 🔍 Contains Duplicate EASY Hash Set Array
expected: True

📝 Problem Statement

Diya gaya integer array nums mein kya koi value 2 ya zyada baar repeat hoti hai? Agar haan, return True. Agar saari values distinct (unique) hain, return False. Bilkul simple — duplicate detection.

🧠 Solving Approach

Hash set use karenge — jisme O(1) lookup hota hai. Array ko ek baar traverse karenge. Har element ke liye check: kya yeh element seen set mein already hai? Agar haan → duplicate mil gaya, return True. Agar nahi → set mein add karke aage badho. Loop khatam hua matlab koi duplicate nahi.

🎯 Core Concepts

Hash SetO(1) average lookup + insert. Memory mein elements ka quick directory.
Early ExitPehla duplicate milte hi return True — pura array scan zaroori nahi.
Single PassEk hi traversal kaafi — har element ek baar dekha jaata hai.

⏱ Complexity

Time: O(n) Space: O(n)
