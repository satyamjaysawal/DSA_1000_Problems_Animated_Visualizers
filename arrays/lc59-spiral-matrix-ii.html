GT LEARN #59 🌀 Spiral Matrix II MEDIUM Matrix · Simulation · O(n²) exp: [[1,2,3],[8,9,4],[7,6,5]]
📝 Problem Statement
Given a positive integer n, generate an n × n matrix filled with elements from 1 to n² in spiral order — start at top-left, go right, then down, then left, then up, shrinking the boundary each time.
🧠 Solving Approach
Maintain 4 boundaries: top, bottom, left, right. Walk along each edge in order (→ ↓ ← ↑), placing numbers 1, 2, 3, .... After completing each edge, shrink that boundary inward. Continue until all boundaries cross.
🎯 Core Concepts
4 Phases per ring: Right (top row), Down (right col), Left (bottom row), Up (left col). Each phase shrinks one boundary.
Why guards before Left/Up? For odd n, the center cell would be filled twice. Check top ≤ bottom and left ≤ right before going Left/Up to avoid double-fill.
Termination: When top > bottom OR left > right — outer while loop exits. Total cells filled = n².
⏱ Time: O(n²) 💾 Space: O(1) extra
