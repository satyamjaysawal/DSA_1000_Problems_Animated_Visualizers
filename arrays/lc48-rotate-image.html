📝 Problem Statement

Given an n × n 2D matrix representing an image, rotate the image by 90 degrees clockwise. Modify the matrix in-place — do NOT allocate another 2D matrix.

🧠 Solving Approach (Transpose + Reverse)

Two-pass O(1)-space trick: Pass 1 — Transpose the matrix (swap matrix[i][j] with matrix[j][i] across the main diagonal). Pass 2 — Reverse each row. Result: 90° clockwise rotation!

🎯 Core Concepts

Transpose: rows ↔ columns (mirror across diagonal) j > i only: Avoid double-swap below diagonal Reverse rows: Final flip → 90° CW Why this works: Transpose + horizontal flip = 90° CW rotation (linear algebra identity)
⏱️ Time: O(n²) 💾 Space: O(1)
