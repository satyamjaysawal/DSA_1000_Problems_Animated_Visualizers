GT LEARN #238 ✖️ Product of Array Except Self MEDIUM Array Prefix/Suffix
expected: [24,12,8,6]

📝 Problem Statement

Diya gaya integer array nums, return karo array answer jisme answer[i] = product of all elements except nums[i]. Constraint: division use mat karo, aur algorithm O(n) time mein chalna chahiye.

🧠 Solving Approach

Two-pass prefix-suffix trick: Pehle left-to-right jaake har position pe "uske left ka product" store karo (yahi answer[i] ka pehla part hai). Phir right-to-left jaake "uske right ka product" running multiply karo. Final answer[i] = leftProduct × rightProduct. O(1) extra space (output array doesn't count).

🎯 Core Concepts

Prefix PassLeft-to-right scan — answer[i] stores product of nums[0..i-1].
Suffix PassRight-to-left scan — multiply answer[i] by product of nums[i+1..n-1].
No DivisionTricky constraint — handles zero values gracefully without divide-by-zero.

⏱ Complexity

Time: O(n) Space: O(1) extra
