GT LEARN #252 📅 Meeting Rooms EASY Sort Greedy Array
intervals= exp: false

📝 Problem Statement

Diye gaye meeting intervals jahan har interval [start, end] hai — determine karo ki ek person saari meetings attend kar sakta hai ya nahi. Return true agar koi overlap nahi, warna false.

🧠 Solving Approach

Pehle intervals ko start time se sort karo. Phir adjacent pairs check karo — agar previous meeting ka end > next meeting ka start, toh overlap hai → return false. Saare pairs clean → return true.

🎯 Core Concepts

Sort by start time: Sirf adjacent pairs check karna kafi hai — non-adjacent overlap automatically catch ho jayega.
Overlap condition: intervals[i-1].end > intervals[i].start means clash. Touch (==) is OK per LeetCode definition.
Early exit: Pehla conflict milte hi false return — no need to check further.
⏱️ Time: O(N log N) 💾 Space: O(1)
