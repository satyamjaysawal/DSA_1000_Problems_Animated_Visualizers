📝 Problem Statement

At a lemonade stand, each lemonade costs $5. Customers stand in queue paying with $5, $10, or $20 bills (one lemonade each). You start with NO change. Return True if you can give every customer correct change in order, else False.

🧠 Solving Approach (Greedy)

Maintain counts of $5 and $10 bills. For each customer, deterministically give change. Key greedy insight: when paying back $15 (for $20), prefer giving $10 + $5 over 3×$5 — save the $5s for future $10 customers!

🎯 Three cases: (1) $5 → just keep it. (2) $10 → must return $5 as change. (3) $20 → return $15 as change: ideally $10+$5, fallback 3×$5.
🪙 Why greedy works: $20 bills are useless as change (no one pays $25). $10 bills only help with $20 customers. $5 bills are most flexible — hoard them. So always burn $10s first when possible.
⏱ Time: O(n) 💾 Space: O(1)
