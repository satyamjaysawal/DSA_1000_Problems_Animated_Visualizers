📝 Problem Statement

Given n non-negative integers representing an elevation map (bar width = 1), compute how much water can be trapped after raining.

🧠 Solving Approach

Two pointers from both ends. Track max height seen from each side. Water at any bar = min(left_max, right_max) − height. Move the smaller side inward each step.

🎯 Core Concepts & Base Case

Base Case: left ≥ right → pointers crossed → return total water
Transition: Process smaller side. If bar < its max → water trapped. If bar ≥ its max → update max.
Key Insight: Water at i = min(left_max, right_max) − height[i]. Two pointers guarantee correct min from the moving side.
Time O(n)Space O(1)
