GT LEARN #218 🌆 The Skyline Problem HARD Heap Sweep Line Sorting
buildings= exp: 7 key points

📝 Problem Statement

Given buildings[i] = [Lᵢ, Rᵢ, Hᵢ] — left edge, right edge, and height of i-th building — return the skyline outline as a list of [x, y] key points. A key point is where the skyline's height changes. Output sorted by x-coordinate. Last point must have y=0 (ground level).

🧠 Solving Approach

Sweep line + max heap — har building ke start aur end ko events mein convert karo, sort karo, fir x-axis pe left-to-right scan karo. Max heap mein active buildings ki heights track karte hain. Jab top-of-heap height change ho, new key point record.

🎯 Core Concepts & Base Case

Events: Start event (L, -H, R) negative height ke saath (heap mein max ke liye). End event (R, 0, 0) just marker.
Lazy deletion: Heap se "expired" buildings ko hatao tabhi jab top pe aaye AND uska end ≤ current x ho.
Key point check: Current max height ≠ last result height → naya skyline point add.
Time O(N log N) Space O(N)
