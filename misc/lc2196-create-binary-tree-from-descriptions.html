descriptions= exp: root=50
📝 Problem Statement

Given a 2D array descriptions where each [parent, child, isLeft] describes a parent-child relationship in a binary tree of unique values. If isLeft=1 child is left, else right. Build the binary tree and return its root.

🧠 Solving Approach

Use a HashMap nodes to store value→TreeNode references (so we can attach children later). Use a Set children to track every value that appears as a child. After processing, the root is the unique node that is NOT in children.

🗺️ HashMap nodes
Lazy node creation — create only if not seen before, then wire up parent/child pointers.
📦 Children Set
Every child value goes here. Whichever node is NOT a child must be the root.
🏆 Root Detection
Loop through all nodes, return the one not present in children.
⏱ Time: O(N) 💾 Space: O(N)
