GT LEARN #193 📝 Tenth Line EASY String / Loop
numLines= exp: "Line 10"

📝 Problem Statement

Given a text file file.txt, print just the 10th line of the file. If the file has fewer than 10 lines, output should be empty (or no output).

🧠 Solving Approach

Iterate through each line keeping a count. Jab count == 10 ho, us line ko store kar lo aur return kar do. Agar loop khatam ho gaya bina 10 reach kiye, empty string return karo.

🎯 Core Concepts

Counter Pattern: count = i + 1 — 1-indexed line numbers track karna.
Early Exit: 10th line milte hi return — baaki lines scan karne ki zaroorat nahi.
Edge Case: Agar n < 10, loop completes without finding → return "".
Time: O(min(n, 10)) Space: O(1)
