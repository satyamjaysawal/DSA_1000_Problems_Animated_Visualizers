📝 Problem Statement

Given an array of integers heights representing the bar heights of a histogram (each bar width = 1), return the area of the largest rectangle that can be formed using contiguous bars.

🧠 Solving Approach

Use a monotonic increasing stack of indices. Jab koi choti bar mile, taller bars ko stack se pop karke unka maximum rectangle calculate karo. Ek sentinel iteration (i = n) at the end with virtual h=0 ensures sab pending bars flush ho jayein.

🎯 Core Concepts & Base Case

Monotonic Stack: Indices in increasing order of height. Naya bar joins only if it's ≥ top.
Pop & Measure: Jab popped bar nikle, uski rectangle = height[top] × width jahan width = i if stack empty, else i - stack[-1] - 1.
Sentinel Trick: range(n+1) mein last iter par h=0 (virtual) — taaki remaining stack bars bhi process ho jayein.
Time O(n) Space O(n)
