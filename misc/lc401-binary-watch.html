📝 Problem Statement

Binary watch mein 4 LEDs upar (hours: 8, 4, 2, 1) aur 6 LEDs neeche (minutes: 32, 16, 8, 4, 2, 1) hote hain. Har LED ek bit hai. Diye gaye turnedOn integer ke liye, saare possible times return karne hain jo watch dikha sakti hai. Hours 0-11 aur minutes 0-59 mein. Format: "H:MM".

🧠 Solving Approach

Sabse simple — brute force enumeration. Har (h, m) pair try karo jahan 0 ≤ h < 12 aur 0 ≤ m < 60. Har pair ke liye popcount(h) + popcount(m) nikalo. Agar yeh sum turnedOn ke barabar hai, toh formatted string list mein add karo. Total 720 combinations, har ek O(1) check.

🎯 Core Concepts

Popcountbin(x).count('1') — number ke binary representation mein 1's count.
Enumeration12 × 60 = 720 combinations — small enough for brute force.
Formatf"{h}:{m:02d}" — minutes ko 2-digit zero-padded format mein.

⏱ Complexity

Time: O(720) = O(1) Space: O(1) result
