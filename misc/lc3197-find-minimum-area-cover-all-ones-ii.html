📝 Problem Statement

Given a 2D binary array grid, find 3 non-overlapping rectangles with non-zero areas (horizontal/vertical sides) such that all the 1s in the grid lie inside these rectangles. Return the minimum possible sum of their areas. Rectangles are allowed to touch.

🧠 Solving Approach

Since rectangles are axis-aligned and non-overlapping, the partition into 3 regions falls into 6 strategies: 2 horizontal cuts, 2 vertical cuts, or 1 horizontal + 1 vertical cut (4 corner variations). For each partition, compute min bounding rect area for each region using helper minArea.

🎯 Core Concepts

Helper: minArea(r1,c1,r2,c2) 6 Partition Strategies Track Global Min
⏱ Time: O(m²·n²·(m+n)) 💾 Space: O(1)
