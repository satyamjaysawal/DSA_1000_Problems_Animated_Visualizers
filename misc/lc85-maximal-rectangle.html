📝 Problem Statement

Given a binary matrix of '0's and '1's, find the largest rectangle containing only 1s and return its area. Rectangle must be axis-aligned (sides parallel to matrix).

🧠 Solving Approach

Har row ko histogram base treat karo — column ke continuous 1s ki count = bar height. Fir har row pe Largest Rectangle in Histogram (LC 84) ka monotonic stack algorithm chalao. Max across all rows = answer.

🎯 Core Concepts & Base Case

Heights array: heights[c] = current row se upar continuous 1s ki count. Agar '0' mile → reset to 0.
Monotonic stack: Indices stored in increasing order of height. Jab choti bar mile, taller bars pop karke unka rectangle calculate karo.
Sentinel trick: Loop i = cols tak (extra), virtual h=0 use karke remaining stack flush — taaki pending bars bhi process ho.
Time O(R·C) Space O(C)
