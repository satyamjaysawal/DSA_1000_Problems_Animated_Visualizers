GT LEARN #195 📝 Transpose File MEDIUM Shell awk String
exp: name alice ryan / age 21 30
📝 Problem Statement
Given file.txt where each line has same number of fields separated by spaces — write a shell command to transpose it. Rows become columns, columns become rows.
🧠 Solving Approach
Classic awk one-liner. Per-line block accumulates each field into s[i] (column-wise array). On first line (NR==1) initialize, otherwise append with space. Final END block prints each s[i] as one transposed line.
🔑 Per-line block
awk runs main {...} block once per input line. NR=current line number, NF=current field count, $i=i-th field.
🪣 Column accumulator
s[i] stores the i-th column being built up. After all lines processed, s[i] = "field1_of_col_i field2_of_col_i ..." — exactly the i-th transposed row.
🏁 END block
END {...} runs after all input consumed. Iterate i=1..NF and print s[i] — output is transposed file.
⏱ Time: O(rows × cols) 💾 Space: O(rows × cols)
🧪 Test Cases
name age / alice 21 / ryan 30 3×2
→ name alice ryan / age 21 30
a b c single row
→ a / b / c

→ x y z

→ 1 3 / 2 4
