GT LEARN #150 🧮 Evaluate Reverse Polish Notation MEDIUM Stack
tokens= exp: 9

📝 Problem Statement

Diya gaya hai ek array tokens jisme Reverse Polish Notation (postfix) mein arithmetic expression hai. Yeh evaluate karke integer result return karo.

Operators: +, -, *, /. Division truncate towards zero hoti hai (jaise int(a/b)).

🧠 Solving Approach

Stack ka use karenge. Har token ke liye: agar number hai toh push karo. Agar operator hai toh top 2 elements pop karo (b first, then a), a OP b calculate karo, result push karo. Final answer = stack[0].

🎯 Core Concepts

Postfix order: Operators apne 2 operands ke baad aate hain. "2 1 +" means 2+1=3.
Pop order matters: First pop = b (right operand), second pop = a (left). Compute a OP b, NOT b OP a.
Integer division: Python ka // floor division use NAHI karo — int(a/b) use karo (truncate toward zero).
Time: O(n) Space: O(n)
