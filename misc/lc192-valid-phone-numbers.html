GT LEARN #192 📝 Valid Phone Numbers EASY String / Regex
preset: exp: 2 valid

📝 Problem Statement

Given a text file file.txt jisme har line pe ek phone number ho sakta hai, sirf valid phone numbers print karo. Ek valid phone number do format mein ho sakta hai:

(xxx) xxx-xxxx  OR  xxx-xxx-xxxx

x = digit (0-9). Length exactly meet karni chahiye, koi extra space ya char nahi.

🧠 Solving Approach

Har line ko regex pattern se match karo. Pattern: ^(\d{3}-|{{{{\(\d{3}\)}}}} )\d{3}-\d{4}$. Agar match hua toh print, warna skip.

🎯 Core Concepts

Anchors: ^ aur $ ensure karte hain ki poori line match ho — partial match nahi chalega.
Alternation: (A|B) — area code do format mein: 123- ya (123) .
Quantifier: \d{3} = exactly 3 digits.
Time: O(n × L) Space: O(1)
