GT LEARN #3635 🎢 Earliest Finish Time for Land & Water Rides II MEDIUM Greedy Sorting Array
expected: 9

📝 Problem Statement

Tourist ko 2 rides karni hain — ek Land aur ek Water. Har ride ke 2 fields hain — start[i] (kab earliest start ho sakti hai) aur duration[i] (kitni der lagti hai). Agar pehli ride t time pe khatam hoti hai aur t ek multiple of 5 hai, toh dusri ride pe transit ke liye +5 minute extra lagte hain. Goal: minimum finish time nikalna jab dono rides exactly ek-ek baar ki jayein.

🧠 Solving Approach

Sirf 2 orders possible hain — Land→Water ya Water→Land. Har order ke liye, hum sabse jaldi finish hone wali first ride pick karein, fir uske finish time t se sabse jaldi finish hone wali second ride. Lekin actually optimal first ride wahi hoti hai jiska start + duration minimum ho — toh sort ya min-pass kaafi hai. Dono orders ka best lekar minimum return karte hain.

🎯 Core Concepts

Greedy PickFirst ride: jiska start+duration sabse chota — woh sabse jaldi free karega slot.
%5 PenaltyAgar t % 5 == 0, transit cost +5 add hoti hai before second ride start.
SymmetryLand→Water aur Water→Land — dono try karke min lo.

⏱ Complexity

Time: O(n + m) Space: O(1)
