📝 Problem Statement

Given an integer array nums and integer k, partition nums into groups of exactly size k where every group has k distinct elements. Return true if such a partition is possible, else false.

🧠 Solving Approach

Two checks: (1) len(nums) must be divisible by k. (2) No element can appear more than numGroups = n/k times — because each group has distinct elements, so an element fits in at most one slot per group.

🎯 Core Concepts

Pigeonhole: If a value appears f times, you need at least f different groups to place all its copies. So f ≤ numGroups is required.
Divisibility: Total elements must split evenly: n % k == 0, otherwise can't form whole groups.
Sufficiency: If both checks pass, partition is always possible (greedy round-robin fill).
Time O(n) Space O(n)
