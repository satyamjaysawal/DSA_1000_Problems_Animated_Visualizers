GT LEARN #194 📝 Word Frequency MEDIUM Shell Pipeline awk/sort/uniq
exp: the 4 / is 3 / sunny 2 / day 1
📝 Problem Statement
Given words.txt with words separated by spaces (multiple per line), write a shell script to output each unique word and its frequency, sorted in descending order of frequency.
🧠 Solving Approach
Classic UNIX pipeline — chain small tools together: cat → tr → sort → uniq -c → sort -nr → awk. Each stage transforms the stream a little. uniq -c needs sorted input — that's why we sort first.
🔧 tr -s ' ' '\n'
Squeeze consecutive spaces and replace each space with newline → one word per line.
📊 sort | uniq -c
uniq only collapses adjacent duplicates → MUST sort first. Output: " count word" (count is left-padded).
🔢 sort -nr
Numeric (-n) reverse (-r) sort by the leading count column → highest frequency first.
🎨 awk '{print $2, $1}'
Swap columns: count became $1, word became $2. Output as "word count".
⏱ Time: O(N log N) 💾 Space: O(N)
