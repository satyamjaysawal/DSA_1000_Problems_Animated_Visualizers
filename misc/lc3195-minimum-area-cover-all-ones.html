GT LEARN #3195 🧱 Min Area to Cover All Ones I EASY Matrix Bounding Box
grid= exp: 6

📝 Problem Statement

Given a 2D binary array grid, return the minimum area of an axis-aligned rectangle that contains all the 1's in the grid. The rectangle's sides must be parallel to the grid axes.

🧠 Solving Approach

Bounding box technique — track min_r, max_r, min_c, max_c across all 1-cells. Final area = (max_r - min_r + 1) × (max_c - min_c + 1). Single O(m·n) pass.

🎯 Core Concepts

Init Trick: Start min_r = m, max_r = -1 (impossible values) so first 1 will always update both.
Bounding Box: Smallest rectangle covering all 1s = top-left at (min_r, min_c), bottom-right at (max_r, max_c).
Area Formula: Width = max_c - min_c + 1, Height = max_r - min_r + 1. The +1 because both endpoints inclusive.
Time O(m·n) Space O(1)
